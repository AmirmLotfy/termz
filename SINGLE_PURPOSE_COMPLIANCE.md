# Single Purpose Compliance - Termz Extension

## Extension Purpose Statement

**Termz** has a single, clearly defined purpose:

> **Analyze legal documents (privacy policies, terms of service, legal agreements) to identify risks, translate legal jargon into plain language, and help users make informed decisions about what they're agreeing to.**

## Purpose Breakdown

### Primary Function
Analyze legal documents using AI to:
1. Detect and flag concerning clauses (data collection, liability waivers, arbitration, etc.)
2. Translate legal jargon into everyday language
3. Provide summaries and key points
4. Create glossaries of legal terms

### Core Activities
All features directly support this purpose:
- **Auto-detection**: Identifies legal document pages as users browse
- **Risk analysis**: Identifies problematic clauses and assigns risk scores
- **Plain language translation**: Converts legal text to understandable language
- **Summaries**: Provides executive summaries, key points, and full breakdowns
- **Glossaries**: Defines legal terms in simple language
- **History**: Saves analysis results for later reference
- **Manual analysis**: Allows users to analyze pasted text or uploaded files

## Feature Alignment

Every feature in Termz serves the core purpose:

| Feature | Purpose Alignment |
|---------|------------------|
| Auto-detection | Identifies legal documents that need analysis |
| Risk assessment | Flags concerning clauses (core analysis purpose) |
| Plain language | Translates jargon (core purpose) |
| Summaries | Makes documents understandable (core purpose) |
| Glossary | Explains legal terms (supports understanding) |
| History | Preserves analysis results for reference |
| Manual analysis | Allows analysis of documents not auto-detected |
| Settings | User preferences for analysis behavior |
| Notifications | Alerts users to detected legal documents |
| File upload | Analyzes offline documents |
| Context menu | Quick access to analysis feature |

## No Unrelated Functionality

Termz does NOT include any features unrelated to legal document analysis:
- ❌ No ad blocking
- ❌ No password management
- ❌ No bookmark management
- ❌ No social media integration
- ❌ No shopping tools
- ❌ No productivity tools unrelated to legal documents
- ❌ No general text analysis (only legal documents)
- ❌ No email or messaging features
- ❌ No file management beyond document upload for analysis

## Permission Alignment

All requested permissions directly serve the single purpose:

| Permission | Purpose Connection |
|-----------|-------------------|
| `sidePanel` | Displays analysis results |
| `tabs` | Accesses pages for detection and analysis |
| `notifications` | Alerts users to detected legal documents |
| `storage` | Saves analysis history and preferences |
| `contextMenus` | Quick access to analysis feature |
| `<all_urls>` | Detects legal documents on any website |

## User Expectations

When users install Termz, they expect:
- ✅ To analyze legal documents
- ✅ To understand legal jargon
- ✅ To identify risks in privacy policies and terms of service
- ✅ To receive plain-language explanations

Users do NOT expect:
- ❌ General text analysis features
- ❌ Non-legal document processing
- ❌ Unrelated productivity tools

## Chrome Web Store Policy Compliance

Termz complies with the Chrome Web Store's Single Purpose Policy:

> "Extensions must have a single purpose that is clear to users. Extensions should do one thing and do it well."

**Termz does one thing**: Analyzes legal documents to help users understand what they're agreeing to.

### Evidence of Compliance

1. **Clear Purpose**: Extension name, description, and all features clearly communicate legal document analysis
2. **Focused Functionality**: Every feature directly supports legal document analysis
3. **No Feature Bloat**: No unrelated tools or functions
4. **Consistent Experience**: All user interactions relate to document analysis
5. **Transparent Intent**: Store listing and documentation clearly state the single purpose

## Conclusion

Termz has a single, well-defined purpose: **to help users understand legal documents by analyzing them for risks and translating jargon into plain language**. Every feature, permission, and user interaction supports this purpose. The extension does not include any unrelated functionality, making it fully compliant with Chrome Web Store's Single Purpose Policy.
